* SRFI 135: Immutable Texts

** by William D Clinger

This repository hosts [[http://srfi.schemers.org/srfi-135/][SRFI 135]]: Immutable Texts, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

This SRFI is in /final/ status.

The full documentation for this SRFI can be found in the [[http://srfi.schemers.org/srfi-135/srfi-135.html][SRFI Document]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-135/][join the SRFI-135 mailing list]] and send your message there.

Thank you.
